## 3. End-to-End 시연 흐름 (예상 5분)

```
[00:00] 업로드 화면
        → 파일 드래그앤드롭: 한샘_리하우스_34평_견적서_샘플.xlsx
        → "한샘리하우스" 브랜드 자동 감지 표시

[00:30] 처리 중 (ProgressBar)
        → PENDING → PROCESSING → 진행률 바 애니메이션
        → 18개 항목 순차 처리 로그 표시 (선택)

[01:00] 분류 결과 요약 카드
        → "13개 자동 완료 / 5개 검토 필요"
        → 자동 완료 항목: 표로 즉시 표시 (공정/신뢰도 포함)

[01:30] 컨펌 루프 화면 (핵심 장면)
        → 카드 1: "인건비" → 사용자가 "철거공사 > 인건비"로 확정
        → 카드 2: "EP도장 2차" → AI 제안 "도장공사" → 사용자 수락
        → 카드 3: "걸레받이 MDF 10T" → 사용자 "목공사"로 확정
        → 카드 4: "기타 잡공사" → 사용자 공종 직접 입력
        → 카드 5: "시스템 선반 W900" → 사용자 "가구공사"로 확정

[03:30] 컨펌 완료 → 자동 export 트리거
        → BidFormatter 실행
        → "한샘리하우스 양식으로 정리된 결과 파일 다운로드" 버튼

[04:00] 결과 파일 다운로드 & 확인
        → 원본 엑셀 양식 유지
        → 공정 분류 + 표준품명 자동 기입
        → 수식 셀 그대로 유지 (변조 없음)

[04:30] 키워드 학습 포인트 강조 (선택)
        → "EP도장 → 도장공사" 패턴이 keyword_dict에 자동 누적
        → 다음 번엔 AI가 바로 처리 (학습 루프 시각화)
```

## 5. 시연 포인트별 강조 멘트

### 포인트 1: 브랜드 자동 감지
> "18년 현장에서 수백 개 브랜드 양식을 다뤄봤습니다. 각 브랜드마다 열 위치, 시트명, 시작 행이 다릅니다. 이걸 매번 수동으로 맞추는 게 기존 방식이었습니다."

### 포인트 2: keyword_dict 즉시 처리 vs Claude API
> "13개는 현장 용어 사전으로 즉시 분류했습니다. 비용도 0원, 시간도 0.1초입니다. AI는 진짜 불명확한 5개에만 투입됩니다."

### 포인트 3: review_flag 컨펌 루프
> "'인건비'는 사람도 모릅니다. 어느 공종의 인건비인지 적혀있지 않으니까요. AI가 자신 없는 항목을 솔직하게 표시하고, 전문가에게 확인을 요청합니다. 이게 18년 경험을 AI에 담은 핵심입니다."

### 포인트 4: 학습 루프
> "사용자가 확정한 'EP도장 = 도장공사' 패턴은 keyword_dict에 자동으로 누적됩니다. 쓸수록 AI가 더 똑똑해집니다."
