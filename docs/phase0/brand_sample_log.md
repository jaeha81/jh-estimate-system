# 브랜드 샘플 분석 기록 (brand_sample_log.md)

> **작성 방법:** 브랜드별 엑셀 견적서 양식을 분석하여 기록

## 분석 체크리스트 (브랜드별 필수 확인 항목)

- [x] 시트 목록 (탭 이름 전부)
- [x] 세부내역 시트 이름
- [x] 세부내역 시트의 데이터 시작 행 번호
- [x] 각 열 위치 (품명/규격/단위/수량/단가/금액/재료비/노무비/경비)
- [x] 행 수 고정 여부
- [x] 병합 셀 존재 여부
- [x] 수식 셀 존재 여부

|---|---|---|

| 2 | 세부내역서 | 세부내역 ← 핵심 |
| 3 | 자재단가표 | 자재 단가 참조표 |
| 4 | 일위대가 | 노무비/경비 산출 근거 |

### 세부내역 시트 상세
- **시트명:** `세부내역서`
- **데이터 시작 행:** 7행 (1~6행: 헤더/타이틀/열 제목)

|---|---|---|

### 구조 특이사항
- 행 수 고정 여부: ✅ 고정 (공종별 행 수 사전 정의)
- 병합 셀 존재: ✅ 있음 (대공종 헤더 행에서 A:B 병합)
- 수식 셀 존재: ✅ 있음 (I열=F+G+H, J열=E×I, 소계/합계 SUM)
- 기타: 대공종 헤더 행은 배경색 회색, 품명 비어있으면 소계 행

## 공통 패턴 분석

### 공통 열 위치
- 품명은 대부분 **B열** (래미안만 C열 — B열이 대공종 분류)
- 금액은 구조에 따라 **G~L열** 범위
- 수량은 **D~F열** 범위

### 브랜드별 주요 차이점
| 브랜드 | 시트명 | 데이터 시작행 | 재료/노무 분리 | 행 고정 | 특이사항 |
|---|---|---|---|---|---|
| 한샘리하우스 | 세부내역서 | 7행 | ✅ (F/G/H) | ✅ | 표준 구조 |
| 래미안인테리어 | 내역서 | 5행 | ❌ (통합) | ✅ | B열 대공종 세로병합 |
| 영림가구 | 견적서 | 10행 | ❌ (통합) | ❌ | 갑지+내역 통합시트 |
| 일반인테리어A | 세부내역 | 4행 | ❌ (통합) | ✅ | A열 대공종 세로병합 |
| 관급양식 | 세부내역서 | 8행 | ✅ (F/G+H/I+J/K) | ✅ | 열 12개, 가장 복잡 |

### 범용 스키마 확정 사항
- 모든 브랜드에서 반드시 존재하는 열: **품명, 수량, 단가, 금액**
- 브랜드마다 다른 열: **재료비/노무비/경비 분리 여부**, **시트명**, **데이터 시작행**
- MVP에서 제외할 열: **재료비/노무비/경비 금액 분리 컬럼** (통합 단가로 처리)
- 핵심 매핑 포인트: `data_start_row`, `item_name` 열 위치, `amount` 열 위치
